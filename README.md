Как пользоваться прогрммой и зачем она нужна: В кабинете стоит шкаф, и для удобства в документации создал визуально удобное приложение с содержимым шкафа. 
Как пользоваться: 1. По нажатию ЛКМ появляется окно с содержимым шкафа, но изменять его нельзя 2. По нажатию ПКМ Происходит переход на сцену с его приближенной версией и там уже есть возможность редактирования.
Есть кнопка 'Экспортировать', которая позволяет экспортировать .txt файлом содержимое шкафа (полки от 1 до 18).
